## Bio Auth Demo App

This project gives an overview of some of the native components available in Titanium. 
It also includes full support for iOS, Android.

### Features
- Labels
- TextFields
- ImageViews
- Bio Authentication (iOS)
- Global Styling
- Mock Data
- Backbone collections
- screen transitions
